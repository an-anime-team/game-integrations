# lua-proton

Simple library to handle proton versions. Unlike `components`, this library is
designed to be more straightforward for end users.

## Integration

Add `lua-proton` package to your integration package inputs:

```json
{
    "inputs": {
        "lua-proton": "https://raw.githubusercontent.com/an-anime-team/game-integrations/refs/heads/master/packages/lua-proton/package.json"
    }
}
```
