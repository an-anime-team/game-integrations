# sophon-tools

Lua bindings to the [sophon-tools](https://dawn.wine/dawn-winery/sophon-tools)
CLI.

## Integration

Add `sophon-tools` package to your integration package inputs:

```json
{
    "inputs": {
        "sophon-tools": "https://raw.githubusercontent.com/an-anime-team/game-integrations/refs/heads/master/packages/sophon-tools/sophon-tools.luau"
    }
}
```
